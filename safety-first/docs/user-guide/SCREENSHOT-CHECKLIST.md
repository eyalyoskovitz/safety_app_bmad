## המרה ל-Word/PDF

לאחר הוספת כל הצילומים, ניתן להמיר את המדריך:

### אפשרות 1: Pandoc (מומלץ)
```bash
pandoc README.md -o user-guide.docx
pandoc README.md -o user-guide.pdf
```

### אפשרות 2: VS Code
1. התקינו תוסף "Markdown PDF"
2. פתחו את README.md
3. לחצו Ctrl+Shift+P → "Markdown PDF: Export (pdf)"

### אפשרות 3: אונליין
1. העתיקו את תוכן הקובץ ל-[Dillinger.io](https://dillinger.io)
2. Export → PDF או HTML
